# 🌊 INNOVENT 2026 - The Liquid Edition

A high-performance landing page for a futuristic tech festival. This project explores the "Liquid" design trend—blending glassmorphism, organic animations, and synthesized audio feedback.

## 🚀 Live Demo
[View Live Site](https://sujalsevak.github.io/FSWD-TASK-2/)

## ✨ Highlights
* **Synthesized Audio:** Uses the Web Audio API to create a "Liquid Pop" sound on every click (no external MP3 files needed!).
* **Liquid Glassmorphism:** Custom CSS filters creating a "blob" background effect that feels alive.
* **Interactive Schedule:** A dynamic grid featuring "Know More" event descriptions and a registration system.
* **Modern Stack:** Built with Tailwind CSS for rapid, responsive UI development.

## 🛠️ Tech Used
* HTML5, Tailwind CSS, JavaScript (ES6+), Web Audio API.
